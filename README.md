### To download all hindi discourses(~78GB):
```
LANG="hindi" python3 api.py
```

### To download all english discourses(~102GB):
```
LANG="english" python3 api.py
```

> [!TIP]
> To serve the files on your local network use [Navidrome](https://www.navidrome.org/).
> 
> Use [Amperfy](https://github.com/BLeeEZ/amperfy) as client on phone or laptop.
