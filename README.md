### To download all hindi discources(~78GB):
```
LANG="hindi" python3 api.py
```

### To download all english discources(~102GB):
```
LANG="english" python3 api.py
```

> [!TIP]
> To server the files on your local network use [Navidrome](https://www.navidrome.org/).
> Use [Amperfy](https://github.com/BLeeEZ/amperfy) as client on phone or laptop.
